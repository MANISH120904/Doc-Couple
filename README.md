## ‚öôÔ∏è Workflow Pipeline

### 1. Synthetic Dataset Creation
* Generated 15‚Äì30 high-fidelity images per character using **OpenArt**.
* Utilized **Flux.1 [dev]** with controlnets to vary poses, lighting, and emotions while retaining facial geometry.

### 2. Identity Locking (Fine-Tuning)
* Curated the dataset to remove artifacts.
* Trained a custom **LoRA (Low-Rank Adaptation)** model on **Civit.ai**.
* **Result:** Ability to generate the specific characters in any scenario (cooking, consulting) without identity loss.

### 3. Image-to-Video Animation
* Selected keyframes generated by the custom LoRA.
* Processed via **Google Veo 3.1** using Image-to-Video (I2V) prompting to dictate temporal actions (e.g., "sip coffee," "turn head").

### 4. Audio & Assembly
* Synthesized voiceovers using **ElevenLabs** to match character personas.
* Final composition, cuts, and overlay assembly in **Canva**.
